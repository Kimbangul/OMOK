# OMOK⚫⚪
## 🔍요약
* 제작 기간 : 2023.06 ~ 2023.07
* 사용 기술스택 : React, TypeScript, Styled-components, Recoil
* 구현 기능 : 오목 칸 생성, 승패 및 무승부 판정, 점수 기록

## 📝설명
토이 프로젝트로 작업한 간단한 오목 게임입니다. <br />
10 ~ 20 사이의 행과 열을 설정하여 오목판을 생성할 수 있으며 <br />
스코어 기록 및 리셋 등 부가 기능을 추가하였습니다.
